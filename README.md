# Expense Tracker Application

## Description
This is a simple expense tracker web application built using Flask.
Users can add expenses and see the total amount.

## Features
- Add daily expenses
- View expense list
- Calculate total expenses

## Tech Stack
- Python
- Flask
- HTML
- CSS

## How to Run the Project
1. Install Flask
   pip install -r requirements.txt

2. Run the application
   python app.py
